Key Project — Military Planning Sandbox (Phoenix Technologies)

A fully offline, encrypted military planning tool built in Unity for Mac and Android. Commanders download any map region by lat/lon bounding box, then plan and simulate operations entirely without internet — in the field, on a real sand table if needed.

  • Offline map system: Users define a lat/lon area to cache map tiles locally. The app reads lat/lon, UTM and MGRS natively — switch coordinate systems without losing position context.
  • Full planning layer: Place troops, vehicles, and standard military symbols on the map. Assign waypoints with conditional movement rules; group entire units — tanks, infantry — for synchronized navigation using NavMesh.
  • Freehand drawing & measurement: Annotate directly on the map with freehand drawing tools, MGRS grid overlay, and precise distance measurement between any two points.
  • Multiplayer with command roles: Mac acts as the session host. A dedicated Director role can spectate every participant individually, switch views freely, and take command-level control over the full session.
  • Xbox Kinect sand-table integration: Kinect sensor scans the physical sand surface during calibration — reads height data in real-time and auto-renders terrain features (rivers, mountains, elevation) on the digital map.
  • Security: End-to-end encrypted sessions throughout.
  • Stack: Unity 3D · C# · NavMesh · Xbox Kinect SDK · Netcode · Newtonsoft.Json · Custom MGRS / UTM / lat/lon coordinate math
